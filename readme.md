# Very Simple RDFa

A very simple primer on how RDFa works and can be used to do cool stuff.

## Usage

Start static server on `/html`

```
❯ npm run start
```

Build with origin set from `.env`

```
❯ node index.js
❯ npm run build
```

Build with Example origin:

```
❯ origin=http://localhost:8888 node index.js
❯ npm run example
```
